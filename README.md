AI Google Ads Anomaly Detector 🚀

Мини-приложение для анализа рекламных кампаний и поиска аномалий с использованием AI.

Описание

Приложение собирает данные по рекламным кампаниям и анализирует их с помощью LLM (OpenAI), выявляя потенциальные проблемы в эффективности.

Примеры аномалий:
- Высокие расходы при отсутствии конверсий
- Неэффективные кампании с большим количеством кликов
- Кампании в статусе PAUSED, но с расходами

Технологии
1. Frontend: React
2. Backend: Node.js (Express)
3. AI: OpenAI API
4. База данных: Prisma (в текущей версии используется mock-данные)

⚠️ Важно

В рамках тестового задания интеграция с Google Ads API замокана (mock).
Это связано со сложностью OAuth2 авторизации и необходимостью реального рекламного аккаунта.
Данные кампаний берутся из локального источника (data.json).

🤖 AI Anomaly Engine

AI анализирует кампании и возвращает результат в формате:
{
  "campaign": "Campaign 2",
  "issue": "No conversions despite high cost",
  "severity": "High"
}

Как запустить проект

1. Клонировать репозиторий
git clone https://github.com/vibe-code-solutions/ai-ads-anomaly-detector.git
cd ai-ads-anomaly-detector

2. Установить зависимости
npm install
cd client
npm install

3. Настроить переменные окружения

Создать файл .env в корне проекта:
OPENAI_API_KEY=your_api_key_here

4. Запуск

Backend:

node server.js

Frontend:

cd client
npm run dev

🔘 Функциональность
- Просмотр списка кампаний
- Отображение ключевых метрик (cost, clicks, conversions)
- AI-анализ аномалий
- Кнопка Run Audit для повторного анализа
- Визуальное выделение критических проблем
